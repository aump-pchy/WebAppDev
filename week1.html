สร้างโปรเจ็กต์ด้วย Vite
  • 1
    เปิด Terminal ใน VS Code

  • 2
    ไปยัง Desktop

  • 3

    เปิด browser → http://localhost:5173
ถ้าไม่เห็นหน้าเว็บ: เช็คว่า Terminal แสดง "Local: http://localhost:5173" — ถ้าไม่มีให้แจ้งครู
Git คืออะไร? — เครื่องมือ Version Control ที่บันทึกประวัติการเปลี่ยนแปลงของโค้ด ถ้าทำพัง ย้อนกลับได้เสมอ

⬛
คำสั่ง Git เบื้องต้น
ใช้ใน Terminal ของ VS Code
  • 1
    ตั้งค่าชื่อและอีเมล (ครั้งแรกครั้งเดียว)
    git config --global user.name "ชื่อของคุณ" git config --global user.email "email@example.com"
  • 2
    เริ่มต้น repository ใหม่
    git init
    ทำในโฟลเดอร์โปรเจ็กต์ — สร้างโฟลเดอร์ .git ซ่อนไว้
  • 3
    ดูสถานะไฟล์ที่เปลี่ยนแปลง
    git status
    ใช้บ่อยที่สุด — ดูว่าไฟล์ไหนเปลี่ยน ไฟล์ไหน staged แล้ว
  • 4
    เพิ่มไฟล์เข้า Staging Area
    # เพิ่มไฟล์เดียว git add ชื่อไฟล์.vue # เพิ่มทุกไฟล์ที่เปลี่ยน git add .
  • 5
    บันทึก commit พร้อมข้อความ
    git commit -m "เพิ่มหน้า login"
    ข้อความ commit ควรบอกว่าทำอะไร — ใช้ภาษาไทยได้
  • 6
    ดูประวัติ commit
    git log --oneline
  • 7
    เชื่อมกับ GitHub และ push
    git remote add origin https://github.com/username/repo.git git branch -M main git push -u origin main
    ทำครั้งแรกครั้งเดียว — ครั้งต่อไปใช้แค่ git push
  • 8
    ดึงโค้ดล่าสุดจาก GitHub
    git pull
🔄
Workflow ที่ใช้บ่อยทุกวัน
# 1. แก้โค้ดเสร็จแล้ว → ดูสถานะก่อน git status # 2. เพิ่มทุกไฟล์ที่เปลี่ยน git add . # 3. commit พร้อมข้อความบอกว่าทำอะไร git commit -m "แก้ bug หน้า login" # 4. push ขึ้น GitHub git push
จำง่ายๆ: status → add → commit → push — ทำซ้ำทุกครั้งที่ทำงานเสร็จ

💡
เว็บแอป vs เว็บไซต์ ต่างกันอย่างไร?
ลักษณะเว็บไซต์ทั่วไปเว็บแอปพลิเคชัน
การโต้ตอบอ่านอย่างเดียวสร้าง/แก้ไข/ลบข้อมูลได้
ฐานข้อมูลไม่จำเป็นมักเชื่อมต่อ database
ผู้ใช้ไม่ต้อง loginมีระบบ login/สิทธิ์
ตัวอย่างบล็อก, WikipediaGmail, ระบบจอง, สต็อก

Frontend — ส่วนที่ผู้ใช้มองเห็น

ทุกสิ่งที่ปรากฏบน browser — ปุ่ม ฟอร์ม สี layout animation ล้วนเป็น Frontend

🟦 HTML — โครงสร้าง🎨 CSS / Tailwind⚡ JavaScript💚 Vue.js

Backend — เบื้องหลังที่มองไม่เห็น

ส่วนที่ทำงานบน server — รับ request ประมวลผล ดึงข้อมูลจาก database ส่งกลับ

🟢 Node.js⚙️ Express.js🔐 JWT Auth☁️ Supabase

ตัวอย่าง: เมื่อ Login เข้า Shopee
  • 1
    Client ส่ง Request
    browser ส่ง POST /api/login พร้อม email+password ไปยัง Server
  • 2
    Server รับและตรวจสอบ
    query ไปยัง Database ว่า email/password ถูกต้องไหม
  • 3
    Database ตอบกลับ
    ส่งข้อมูล user กลับมาให้ Server
  • 4
    Server ส่ง Response
    ส่ง JWT token กลับมา (สำเร็จ) หรือ error (ไม่สำเร็จ)
  • 5
    Client แสดงผล
    Browser redirect ไปหน้า dashboard
