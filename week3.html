HTML Semantic Elements

เขียน HTML ให้มีความหมาย — ไม่ใช้แต่ <div>

🤔ทำไมต้องใช้ Semantic Tag?
❌ แบบเดิม — div ทุกอย่าง
<!-- browser ไม่รู้ว่าแต่ละส่วนคืออะไร --> <div class="header">...</div> <div class="nav">...</div> <div class="main">...</div> <div class="footer">...</div>
✅ แบบ Semantic — บอกความหมาย
<!-- browser และ Google รู้ทันทีว่าคืออะไร --> <header>...</header> <nav>...</nav> <main>...</main> <footer>...</footer>
💡 Semantic HTML ช่วย Google จัดอันดับเว็บดีขึ้น และทำให้โค้ดอ่านได้ง่ายกว่ามาก — ไม่ต้องดูที่ class name อีกต่อไป
🔖Tags Semantic ที่ต้องรู้
Tagใช้สำหรับตัวอย่าง
<header>ส่วนหัวของหน้าหรือ sectionโลโก้, ชื่อเว็บ
<nav>แถบ Navigation — ลิงก์เมนูเมนูด้านบน
<main>เนื้อหาหลัก — มีได้แค่ 1 ต่อหน้าพื้นที่เนื้อหาหลัก
<section>แบ่งเนื้อหาเป็นหัวข้อHero, About, Contact
<article>เนื้อหาที่ standalone ได้บทความ, Product Card
<footer>ส่วนท้ายของหน้าCopyright, ลิงก์ท้าย
⚠️ Semantic tag ไม่มีสไตล์พิเศษ — แสดงผลเหมือน div เลย แต่มีความหมายที่ browser เข้าใจ
🗂โครงสร้างหน้าเว็บ — มองเห็นภาพ
<header> โลโก้ + ชื่อเว็บ
<nav> เมนู Home | About | Contact
<main>
<section> Hero
<section> Cards

CSS Flexbox

จัดตำแหน่ง element ได้ง่ายด้วยคำสั่งไม่กี่บรรทัด

🧠Flexbox คืออะไร?

ก่อนมี Flexbox การจัดตำแหน่ง element ทำได้ยากมาก

Flexbox คือระบบ layout ที่ทำให้ items ข้างในกล่องจัดเรียงตัวเองได้อัตโนมัติ — จัดกลาง, จัดชิดขอบ, เว้นระยะ — ด้วย CSS ไม่กี่บรรทัด

/* เปิดใช้ Flexbox */ display: flex; /* แค่นี้ ลูกข้างในจะเรียง */ /* เป็นแถวนอนแล้ว ✓ */
💡 กล่องที่ใส่ display:flex เรียกว่า Flex Container — ของข้างในเรียกว่า Flex Items
↔️แกน Main Axis กับ Cross Axis — จำให้ขึ้นใจ
🎯 ง่ายที่สุด: Main axis = แกนที่ items วิ่ง | Cross axis = แกนตั้งฉาก กับ main axis เสมอ
flex-direction: row (ค่าเริ่มต้น)
main → →
items เรียงซ้าย → ขวา
flex-direction: column
main ↓
items เรียงบน → ล่าง
justify-content
จัดตำแหน่งบน Main Axis (แกนที่ items วิ่ง)
justify-content: flex-start; /* ชิดซ้าย */ justify-content: center; /* กลาง */ justify-content: flex-end; /* ชิดขวา */ justify-content: space-between; /* เว้นขอบ */
align-items
จัดตำแหน่งบน Cross Axis (แกนตั้งฉาก)
align-items: flex-start; /* ชิดบน */ align-items: center; /* กลาง */ align-items: flex-end; /* ชิดล่าง */ align-items: stretch; /* ยืดเต็ม */
