Workflow — ดาวเทียมสู่เส้นทางปลอดภัย
ข้อมูลจาก 3 แหล่ง Open Data ถูกประมวลผลผ่าน ML และ Pathfinding Algorithm เพื่อแนะนำเส้นทางก่อนเจ้าหน้าที่ออกภาคสนาม
01
รับจุดความร้อนจาก NASA FIRMS
ดึงข้อมูล hotspot VIIRS SNPP ทุก 3 ชั่วโมง ครอบคลุมพื้นที่ป่าทั่ว จ.เลย ความละเอียด 375 เมตร พร้อมค่า FRP (Fire Radiative Power) บอกความรุนแรงของไฟ
NASA FIRMS VIIRS API
02
วิเคราะห์สภาพอากาศ Real-time
ดึงทิศทางลม ความเร็วลม ความชื้นสัมพัทธ์ จาก Open-Meteo API แบบ real-time ข้อมูลเหล่านี้เป็นปัจจัยหลักในการคำนวณว่าไฟจะลามไปทิศทางใด
Open-Meteo Weather API
03
AI พยากรณ์การลามไฟ (ML Core)
Random Forest Model ทำนาย risk level ทุก grid ขนาด 500m โดยพิจารณา ทิศลม ความลาดชัน (SRTM DEM) และประเภทพืชพรรณ — train บน Synthetic Dataset ที่สร้างจาก Rothermel fire spread equations แก้ปัญหาการขาด historical data
Random Forest · scikit-learn
04
คำนวณเส้นทางปลอดภัย A*
A* Algorithm ค้นหาเส้นทางบน risk map โดยให้ cost สูงกับ grid ที่ ML ทำนายว่าเสี่ยงสูง ผลลัพธ์คือเส้นทางที่ผ่านโซนอันตรายน้อยที่สุดสำหรับเจ้าหน้าที่
A* Pathfinding · NetworkX
05
แสดงผลบน Web Dashboard
แผนที่ interactive พร้อม fire risk heatmap เส้นทางแนะนำ และ event log real-time เจ้าหน้าที่ใช้วางแผนก่อนออกปฏิบัติการ ไม่ต้องรอข้อมูล real-time จาก field
Leaflet.js · React · Flask

ML Engine

Routing Engine

NetworkX Flask API
Visualization
Web dashboard แผนที่ interactive พร้อม fire risk overlay เส้นทางปลอดภัย event log และ toggle layer — ใช้งานได้บน browser ทุก device

Roadmap
จาก Proof of Concept ในงานแข่ง สู่ระบบที่ใช้งานได้จริงกับเจ้าหน้าที่ป่าไม้ จ.เลย
Phase 1 — แข่งขัน
Proof of Concept · Hackathon Demo
สร้าง Synthetic Dataset จาก Rothermel model · Train Random Forest · คำนวณ A* บนแผนที่ จ.เลย จริง · Demo ในงาน Coding Thailand 2026 ขอนแก่น
Phase 2 — Validate
ทดสอบกับข้อมูลจริง จ.เลย
รวบรวม historical fire data จากกรมอุทยานฯ · Validate model กับเหตุการณ์จริงในอดีต · ปรับ parameter ให้เหมาะกับ terrain จ.เลย · ทดสอบกับเจ้าหน้าที่จริง
Phase 3 — Deploy
ส่งมอบระบบใช้งานจริง
ส่งมอบให้หน่วยงานป่าไม้และอุทยาน จ.เลย · เพิ่ม Mobile App สำหรับเจ้าหน้าที่ภาคสนาม · ขยายครอบคลุมพื้นที่ป่าทั่วภาคอีสาน
