ชิ้นงาน: Profile Card ด้วย Vue

สร้างหน้าแนะนำตัวที่ใช้ ref() เก็บข้อมูล และ v-bind เชื่อมกับ Tailwind — ทำในโปรเจ็กต์ Vite+Vue จริง

📋 โจทย์

สร้าง Profile Card ในไฟล์ App.vue ที่ข้อมูลทั้งหมดมาจาก ref() — ไม่ hardcode ลงใน template โดยตรง

⚠️ข้อมูลทุกอย่างต้องอยู่ใน ref() ใน script เท่านั้น — ห้ามพิมพ์ชื่อหรือข้อมูลตรงๆ ใน template
⭐ เลือกระดับ

Profile ง่ายๆ
แสดงข้อมูลจาก ref() ในหน้าแนะนำตัว
  • ref() อย่างน้อย 4 ตัว (ชื่อ, อายุ, แผนก, งานอดิเรก)
  • แสดงด้วย {{ }} ครบทุก ref
  • จัด layout ด้วย Tailwind
  • มีสีพื้นหลัง card สวยงาม
ระดับ 2 · กลาง
Dynamic Card
ทุกอย่างของระดับ 1 + เพิ่ม v-bind
  • :class bind สี theme จาก ref()
  • :src bind รูปภาพจาก URL ref()
  • :href bind ลิงก์ GitHub ref()
  • ใช้ template literal ใน ref อย่างน้อย 1 จุด

  • แต่ละ card มีธีมสีต่างกันจาก ref()
