HTML คืออะไร?

HTML (HyperText Markup Language) คือภาษาที่บอก browser ว่าหน้าเว็บมีอะไรบ้าง — ข้อความอยู่ที่ไหน รูปภาพอยู่ที่ไหน ปุ่มอยู่ที่ไหน HTML ไม่ใช่ภาษา programming — เป็นแค่ภาษาสำหรับ "จัดโครงสร้าง"

โครงสร้างพื้นฐานของ HTML
<!-- โครงสร้างทุกไฟล์ HTML ต้องมีนี้ --> <!DOCTYPE html> <html lang="th"> <head> <meta charset="UTF-8"> <!-- ภาษาไทยไม่เพี้ยน --> <meta name="viewport" content="width=device-width"> <title>ชื่อแท็บ</title> <!-- Tailwind CDN --> <script src="https://cdn.tailwindcss.com"></script> </head> <body> <!-- เนื้อหาทั้งหมดอยู่ในนี้ --> <h1>สวัสดี World</h1> </body> </html>
จำง่ายๆ: <head> = ส่วนหัว (ข้อมูล, script) · <body> = ส่วนเนื้อหาที่เห็นบนหน้าจอ
💡
Tag คืออะไร?

HTML ใช้ "Tag" เพื่อบอก browser ว่าแต่ละส่วนคืออะไร Tag ส่วนใหญ่มีแบบเปิดและปิด

<h1>ข้อความ</h1> <!-- เปิด/ปิด --> <img src="photo.jpg"> <!-- ปิดตัวเอง (self-closing) -->
🔖
Tags ที่ใช้บ่อยในงานจริง
Tagใช้สำหรับตัวอย่าง
<h1>–<h6>หัวข้อ (h1 ใหญ่สุด)ชื่อหน้า, หัวข้อบทความ
<p>ย่อหน้าข้อความคำอธิบาย, เนื้อหา
<div>กล่องจัดกลุ่ม (ไม่มีความหมายเฉพาะ)wrapper, container
<span>จัดกลุ่มแบบ inlineไฮไลต์คำ, badge
<img>รูปภาพsrc = path รูป, alt = คำอธิบาย
<a>ลิงก์href = URL ปลายทาง
<button>ปุ่มกดsubmit form, trigger action
<input>ช่องกรอกข้อมูลtype="text", type="email"
<ul> <li>รายการ (bullet)เมนู, รายการสินค้า
<nav>แถบ navigationเมนูด้านบนหน้าเว็บ
<section>ส่วนเนื้อหาHero, About, Contact section

Tailwind CSS คืออะไร?

แทนที่จะเขียน CSS เอง Tailwind มี utility class สำเร็จรูปให้ใช้เลย — ใส่ class ลงใน HTML ก็สวยได้ทันที

🆚 CSS ปกติ vs Tailwind
CSS ปกติ
.my-button { background-color: blue; color: white; padding: 8px 16px; border-radius: 8px; font-weight: bold; } <button class="my-button"> กดเลย </button>
Tailwind
<button class="bg-blue-500 text-white px-4 py-2 rounded-lg font-bold"> กดเลย </button>
ผลลัพธ์เหมือนกัน แต่ Tailwind ไม่ต้องตั้งชื่อ class เอง ไม่ต้องสลับไฟล์ CSS — เขียนได้เลยใน HTML
⚡
เพิ่ม Tailwind ในไฟล์ — 1 บรรทัด
<head> <!-- เพิ่มบรรทัดนี้ใน <head> --> <script src="https://cdn.tailwindcss.com"></script> </head>
CDN คืออะไร? — โหลด Tailwind จากอินเทอร์เน็ตโดยตรง ไม่ต้องติดตั้งอะไรเพิ่ม เหมาะสำหรับเรียนและ prototype

Typography — จัดสไตล์ข้อความ
ขนาดตัวอักษร — text-{size}
text-xs → ข้อความเล็กมาก
text-sm → ข้อความเล็ก
text-base → ข้อความปกติ
text-lg → ข้อความใหญ่
text-2xl → ข้อความใหญ่มาก
text-4xl → หัวข้อใหญ่
น้ำหนัก font-{weight}
font-normal ข้อความปกติ
font-medium กึ่งหนา

การจัดชิด text-{align}
text-left ชิดซ้าย
text-center กึ่งกลาง
text-right ชิดขวา
📐
Spacing — p (padding) และ m (margin)
กฎจำง่าย: p = padding (พื้นที่ข้างใน) · m = margin (พื้นที่ข้างนอก) · ตัวเลข 1 = 4px, 2 = 8px, 4 = 16px, 8 = 32px
ทิศทาง Spacing
p-4 → ทุกด้าน px-4 → ซ้าย-ขวา py-4 → บน-ล่าง pt-4 → บน (top) pb-4 → ล่าง (bottom) pl-4 → ซ้าย (left) pr-4 → ขวา (right)
m-4 → ทุกด้าน mx-4 → ซ้าย-ขวา my-4 → บน-ล่าง mt-4 → บน mb-4 → ล่าง mx-auto → กึ่งกลาง space-y-4 → ระหว่าง child
ตัวอย่างเปรียบเทียบ

สี — bg-{color}-{shade} และ text-{color}-{shade}
Pattern จำง่าย: ชื่อสี + เลข shade (100=อ่อน → 900=เข้ม) เช่น bg-blue-500, text-red-600, bg-gray-100
👆 คลิกสีเพื่อดูชื่อ class
ตัวอย่างการใช้งาน
<div class="bg-blue-500 text-white p-4"> กล่องสีฟ้า ตัวอักษรขาว </div> <p class="text-gray-600">ข้อความสีเทา</p> <div class="bg-green-100 text-green-800 p-3"> แจ้งเตือนสำเร็จ </div>

Width & Height — w-{size} และ h-{size}
Width classes ที่ใช้บ่อย
w-1/4
w-1/2
w-3/4
w-full
Width ที่ใช้บ่อย
w-full → 100% w-1/2 → 50% w-1/3 → 33% w-auto → อัตโนมัติ max-w-sm → สูงสุด 384px max-w-md → สูงสุด 448px max-w-lg → สูงสุด 512px max-w-screen-md → 768px
Height ที่ใช้บ่อย
h-screen → 100vh h-full → 100% h-auto → อัตโนมัติ h-16 → 64px h-32 → 128px min-h-screen → ขั้นต่ำ 100vh
mx-auto + max-w-md = กล่อง centered กว้างพอดี — ใช้บ่อยมากกับ layout หลักของหน้า
