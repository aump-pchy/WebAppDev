⚙️ Comparison operators — ใช้บ่อยใน v-if

02

v-else + v-else-if + v-show

แยกแสดงเนื้อหาตามเงื่อนไขหลายกรณี

🔀 v-if / v-else-if / v-else
<!-- เหมือน if / else if / else --> <div v-if="score >= 80"> 🏆 ยอดเยี่ยม </div> <div v-else-if="score >= 60"> ✅ ผ่าน </div> <div v-else> ❌ ไม่ผ่าน </div>
คะแนน (ลากปรับ)
score = 75
✅ ผ่าน
👁 v-show — ซ่อนแต่ยังอยู่ใน DOM
<!-- v-if: ลบออกจาก DOM --> <p v-if="show">ข้อความ</p> <!-- v-show: display:none แทน --> <p v-show="show">ข้อความ</p>
เมื่อไหร่ใช้อะไร?
v-if — ซ่อนนานๆ หรือเนื้อหาหนัก
→ toggle ไม่บ่อย

v-show — toggle บ่อยๆ
→ เช่น dropdown, accordion
💡สำหรับผู้เรียน: ใช้ v-if ไปก่อนได้เลย — v-show ค่อยใช้เมื่อเจอ performance issue

Array + v-for — render รายการ

เก็บข้อมูลหลายชิ้นใน array แล้ว render ออกมาทีเดียว

⚡ JS Concept — Array คืออะไร
Array = กล่องที่เก็บข้อมูลได้หลายชิ้น เรียงกันตามลำดับ แต่ละชิ้นมี index เริ่มจาก 0

// ประกาศ array const menus = ref(['ลาเต้', 'อเมริกาโน่', 'มัทฉะ']) // เพิ่ม item menus.value.push('โกโก้') // ดูจำนวน menus.value.length // 4
🔁 v-for — loop ผ่านทุก item
<!-- ใน script --> const menus = ref([ '☕ ลาเต้', '☕ อเมริกาโน่', '🍵 มัทฉะ', ]) <!-- ใน template --> <ul> <li v-for="menu in menus" :key="menu"> {{ menu }} </li> </ul>

const menus = ref([ { name: 'ลาเต้', price: 80, available: true }, { name: 'มัทฉะ', price: 90, available: false }, ]) <div v-for="item in menus" :key="item.name"> {{ item.name }} - {{ item.price }}฿ </div>

04

:key — ทำไมต้องมี

บอก Vue ว่าแต่ละ item ใน list คือตัวไหน

🔑 :key ช่วยอะไร?
💡Vue ใช้ :key เพื่อ track ว่าแต่ละ item คือตัวไหน เมื่อ array เปลี่ยน Vue จะ update เฉพาะส่วนที่เปลี่ยนโดยไม่ต้อง re-render ทั้งหมด
❌ ไม่มี :key — Vue สับสน <li v-for="item in items">{{ item }}</li> ✅ มี :key — Vue รู้ว่าแต่ละตัวคือใคร <li v-for="item in items" :key="item.id">{{ item.name }}</li>
⚠️:key ต้องเป็นค่า unique ไม่ซ้ำกัน ในแต่ละ list — ใช้ id หรือ name ที่ไม่ซ้ำ อย่าใช้ index ถ้า list อาจเรียงใหม่

Workshop — หน้าเมนูร้านอาหาร

สร้าง App.vue ที่มี array ของเมนูอย่างน้อย 4 รายการ แล้ว render ด้วย v-for และใช้ v-if แสดง badge ตามเงื่อนไข

1
พื้นฐาน: สร้าง array ของเมนู (name, price, available) และแสดงด้วย v-for — ถ้า available = false ให้แสดงข้อความ "หมดแล้ว" ด้วย v-if
2
กลาง: เพิ่ม field category (เครื่องดื่ม/ของหวาน) และ isBestseller — แสดง badge ด้วย v-if ให้ถูกต้องตาม field นั้น
3
ท้าทาย: เพิ่มปุ่ม filter (ทั้งหมด / เครื่องดื่ม / ของหวาน) — เมื่อกดให้แสดงเฉพาะ category นั้น และแสดงจำนวนรายการที่กรองได้
