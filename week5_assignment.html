Week 5 v-if + v-for ⏱ 1–1.5 ชั่วโมง 👤 เดี่ยว

ชิ้นงาน: หน้าเมนูร้านอาหาร

ใช้ v-for แสดงรายการเมนูจาก array และ v-if แสดง badge ต่างๆ ตามเงื่อนไขของแต่ละเมนู

📋 โจทย์

สร้างหน้าเว็บร้านอาหารที่มี array ของเมนู — แต่ละเมนูมี property ของตัวเอง แล้วแสดงผลด้วย v-for พร้อม badge ต่างๆ ด้วย v-if

⚠️ข้อมูลเมนูทั้งหมดต้องอยู่ใน ref([]) ใน script — ห้าม hardcode ชื่อเมนูตรงๆ ใน template
⭐ เลือกระดับ
ระดับ 1 · พื้นฐาน
รายการเมนูง่ายๆ
แสดงเมนูจาก array พร้อมบอกว่าหมดหรือไม่
  • array เมนูอย่างน้อย 4 รายการ
  • แต่ละเมนูมี name, price, available
  • v-for แสดงทุกเมนู มี :key
  • v-if แสดงข้อความ "หมดแล้ว" ถ้า available = false
  • จัด layout ด้วย Tailwind
ระดับ 2 · กลาง
Badge หลายแบบ
ทุกอย่างของระดับ 1 + badge หลายประเภท
  • เพิ่ม field: category, isBestseller, isNew
  • badge "⭐ ขายดี" ถ้า isBestseller = true
  • badge "🆕 ใหม่" ถ้า isNew = true
  • badge "❌ หมด" ถ้า available = false
  • card ของเมนูที่หมดให้จางลง (opacity)
ระดับ 3 · ท้าทาย
Filter + สถิติ
ทุกอย่างของระดับ 2 + filter เมนู
  • ปุ่ม filter: ทั้งหมด / เครื่องดื่ม / ของหวาน
  • v-if กรองแสดงเฉพาะ category ที่เลือก
  • แสดงจำนวนรายการที่กรองได้ ("3 รายการ")
  • ปุ่มที่เลือกอยู่มีสีต่างจากปุ่มอื่น (:class)
  • แสดงสถิติ: มีทั้งหมด / ขายดี / หมดแล้ว

<template> <div class="min-h-screen bg-orange-50 p-8"> <h1 class="text-3xl font-bold text-orange-800 mb-2">☕ {{ shopName }}</h1> <p class="text-orange-600 mb-8">เมนูทั้งหมด {{ menus.length }} รายการ</p> <div class="grid grid-cols-2 gap-4 md:grid-cols-4"> <!-- TODO: เพิ่ม v-for และ v-if ตรงนี้ --> <div v-for="item in menus" :key="item.id" class="bg-white rounded-xl shadow p-4 relative"> <!-- badge ถ้าหมด --> <span v-if="!item.available" class="absolute -top-2 -right-2 bg-red-100 text-red-700 text-xs font-bold px-2 py-0.5 rounded-full border border-red-300"> ❌ หมด </span> <div class="text-3xl mb-2">{{ item.emoji }}</div> <div class="font-semibold text-gray-800">{{ item.name }}</div> <div class="text-orange-600 font-bold mt-1">{{ item.price }}฿</div> </div> </div> </div> </template> <script setup> import { ref } from 'vue' const shopName = ref('ชื่อร้านของคุณ') const menus = ref([ { id:1, name:'ลาเต้', emoji:'☕', price:80, available:true }, { id:2, name:'มัทฉะ', emoji:'🍵', price:95, available:false }, { id:3, name:'โกโก้', emoji:'🍫', price:75, available:true }, { id:4, name:'เค้กช็อก', emoji:'🍰', price:85, available:true }, ]) </script>
💡 Hints ที่อาจช่วยได้
🏷 Badge styles
ขายดี: bg-yellow-100 text-yellow-800 border-yellow-300
ใหม่: bg-blue-100 text-blue-800 border-blue-300
หมด: bg-red-100 text-red-700 border-red-300
🔘 ปุ่ม filter active
:class="selected==='drink' ? 'bg-orange-500 text-white' : 'bg-white'"
เมื่อกด: selected = 'drink'
👁 Card จางเมื่อหมด
:class="!item.available ? 'opacity-50' : ''"
หรือ :class="{'opacity-50': !item.available}"

menus.value.filter(m => m.isBestseller).length
