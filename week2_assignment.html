Week 2 ⏱ 1 – 1.5 ชั่วโมง 👤 เดี่ยว

ชิ้นงาน: สร้างหน้าเว็บ "คาเฟ่ของฉัน"

HTML เพียวๆ + Tailwind CDN · ไม่ต้องใช้ npm หรือ Vite · เปิดใน browser ได้เลย

📋 โจทย์

สร้างหน้าเว็บสำหรับ คาเฟ่สมมติ ชื่อใดก็ได้ที่ตัวเองตั้ง โดยใช้ HTML และ Tailwind CSS เพียงอย่างเดียว — ไม่เขียน CSS เอง

ใช้รูปภาพจาก https://picsum.photos/400/300 (random photo) หรือ https://placehold.co/400x300/3B82F6/white?text=Coffee (placeholder) แทนรูปจริงได้เลย
⭐ เลือกระดับที่ท้าทายตัวเอง
ระดับ 1 · พื้นฐาน
หน้าเมนูเดี่ยว
สร้างหน้าแสดงเมนูคาเฟ่อย่างน้อย 4 รายการ
  • มีชื่อร้านและคำอธิบายสั้น
  • เมนู 4 อย่าง มีชื่อ + ราคา
  • ใช้สี background และ text ต่างกัน
  • มี padding/margin ไม่ให้แน่น
  • ข้อความอ่านง่าย font สมเหตุสมผล
ระดับ 2 · กลาง
หน้าเว็บคาเฟ่ครบ
ทำทุกอย่างของระดับ 1 + เพิ่มส่วนต่อไปนี้
  • Navbar มีชื่อร้านและลิงก์เมนู 3 อย่าง
  • Hero section มีหัวข้อใหญ่ + ปุ่ม CTA
  • เมนู 6 อย่าง จัดเป็น Card มีรูปภาพ
  • Footer มีชื่อร้านและที่อยู่สมมติ
  • ใช้ shadow และ rounded บน card
ระดับ 3 · ท้าทาย
Landing Page เต็ม
ทำทุกอย่างของระดับ 2 + เพิ่มความซับซ้อน
  • Badge "bestseller" หรือ "new" บน card บางอัน
  • Section "เกี่ยวกับเรา" มีรูป + ข้อความ 2 คอลัมน์
  • Hover effect บนปุ่มและ card (hover: classes)
  • Responsive: มือถือ 1 คอลัมน์ / จอใหญ่ 3 คอลัมน์
  • Gradient สีบน Hero หรือ Navbar

🚀 Starter Code — คัดลอกไปเริ่มได้เลย
เปิด VS Code → สร้างไฟล์ cafe.html → วางโค้ดด้านล่าง → บันทึก → ดับเบิลคลิกเปิดใน Safari
<!DOCTYPE html> <html lang="th"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>คาเฟ่ของฉัน</title> <!-- Tailwind CDN --> <script src="https://cdn.tailwindcss.com"></script> </head> <body class="bg-gray-50"> <!-- TODO: สร้างหน้าเว็บคาเฟ่ของคุณที่นี่ --> <!-- ตั้งชื่อร้านเอง เช่น "Morning Brew", "ดอยช้าง คาเฟ่", "Lazy Cat Coffee" --> <h1 class="text-3xl font-bold text-center p-8 text-amber-800"> ☕ คาเฟ่ของฉัน </h1> <!-- เพิ่มเนื้อหาต่อจากนี้... --> </body> </html>
💡 Class ที่อาจมีประโยชน์
🧱 Layout
max-w-4xl mx-auto → กล่องกลางหน้า
grid grid-cols-3 gap-6 → 3 คอลัมน์
flex items-center justify-between → Navbar
flex flex-col gap-4 → stack ลง
🎨 สีและตัวอักษร
text-amber-800 → สีกาแฟ
bg-amber-50 → พื้นหลังครีม
text-gray-500 → คำอธิบายเบาๆ
font-bold text-2xl → หัวข้อใหญ่
🃏 Card
bg-white rounded-xl shadow-md
overflow-hidden → รูปไม่ล้น card
object-cover w-full h-48 → รูปเต็มกว้าง
p-4 → padding เนื้อหาใน card
✨ ระดับ 3 เพิ่มเติม
hover:shadow-xl hover:scale-105
transition duration-200 → animation
bg-gradient-to-r from-amber-400 to-orange-500
sm:grid-cols-1 md:grid-cols-3 → responsive

เนื้อหาครบถ้วน มีชื่อร้าน, เมนูครบตามระดับที่เลือก, ข้อมูลถูกต้องอ่านได้ 25
การใช้ Tailwind ใช้ utility class ได้ถูกต้อง ไม่เขียน CSS เอง 25
ความสวยงาม สีสอดคล้อง, spacing เหมาะสม, อ่านง่าย, ดูเป็นมืออาชีพ 20
